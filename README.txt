날씨 정보 웹 스크래핑

이 프로그램은 날씨 정보를 실시간으로 스크래핑하여 웹 페이지에 표시하는 기능을 제공합니다.

-사용 방법
1. 프로그램을 실행합니다.
2. 웹 브라우저가 열리고, 날씨 정보가 표시된 웹 페이지가 자동으로 로드됩니다.
3. 15초이내에 정보를 찾고싶은 지역을 검색후 기다립니다.
4. 웹 페이지 실행이 완료되면, 현재 날씨 정보 및 날씨에대한 요소 및 총평이 표시됩니다.
5. 마지막으로 페이지를 보고 오늘의 옷차림,우산,미세먼지등 정보를 확인하고 외출시 참고하시면됩니다.

-주의 사항
1.Python3 와 Anaconda 둘중 환경변수를 체크후 원하는 환경의 path(환경변수)를 상단에둡니다.
2.이 프로그램은 Selenium 라이브러리를 사용하므로, 실행하기 전에 해당 라이브러리가 설치되어 있어야 합니다.
  pip install selenium
3.또한, Chrome 브라우저가 있어야하므로, Chrome을 설치하시고 이용해야합니다.
4.프로그램 실행 중에는 인터넷 연결이 필요합니다.

라이선스
이 프로그램은 오픈 소스입니다. 자유롭게 사용, 수정, 배포할 수 있습니다.
